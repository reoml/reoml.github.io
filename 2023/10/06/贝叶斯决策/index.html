$$ 贝叶斯决策=贝叶斯+决策$$

贝叶斯

贝叶斯的思想

一言以蔽之，就是修正自身对于知识的认识的过程。

在最开始，人对世界有一些基础主观认识（贝叶斯里面叫总体密度$p(x)$ ），
当一个新的事物或是知识被人观察到， 人就会去寻找与这个知识或事物有关的经验，这种经验积累出来知识人们约定俗成的将它们看作最初始的客观知识（贝叶斯里面叫先验知识$p(\omega)$）。
随后人们会基于自己对世界的主观认识来对客观知识进行修正或者改造（贝叶斯里面叫条件概率$p(x|\omega)$）。
当整个过程完成之后，人们会形成对事物或知识的一种新的认知（贝叶斯里面叫后验概率$p(\omega|x)$）。
那么，这个过程是怎么实现的呢？亦或者说这些东西之间是有什么关系呢？
贝叶斯告诉我们有这样一个式子，能用简单的乘除方式将这个过程以及彼此的联系简洁的表达出来。
$$P(\omega_i | x) = \frac{p( x \space ,\space \omega_i )}{p(x)} =\frac{p( x \space |\space \omega_i ) \space p(\omega_i )}{p(x)} \space \space \space \space \space \space \space \space \space \space \space \space \space (1)$$
1.这个式子告诉我们，事物的客观知识作用（乘）在我们的认知过程中，当我们以我们主观的世界观为基础（除在分母）来看待这个知识的时候，我们会形成对事物的新理解。
2.其次这个式子还告诉我们，如果我们对事物进行认知的时候，处于一种完全客观的状态下（不以我们的世界观或者认知观为基础），我们将对事物毫无新的认知。因为如果这样成立，则$x$与$\omega$ 互相独立则有如下：
$$P(\omega_i | x) =\frac{p( x \space |\space \omega_i ) \space p(\omega_i )}{p(x)} =\frac{p( x \space ,\space \omega_i ) \space p(\omega_i )}{p(x)}
=\frac{p( x )\space p(\omega_i )}{p(x)}
=p( \omega_i) \space \space \space \space \space \space \space \space \space \space \space \space \space (1)$$
这种揭示人类认知过程的理论被应用到机器学习应当是自然而然的。
